Projeto e Empresa

A equipe desenvolveu um sistema para a empresa Free Soul Food em duas plataformas, web e mobile. O sistema é uma rede de compartilhamento de motoboys, onde os restaurantes cadastrados solicitam uma entrega e o motoboy mais próximo recebe uma notificação para aceitar ou recusar a entrega, se durante o percurso o motoboy receber outra solicitação e esta for aceita, o sistema recalcula a rota para conseguir adicionar os pontos da nova entrega.

A plataforma web foi desenvolvida especialmente para os restaurantes cadastrados, onde os mesmos podem cadastrar entregas que precisam serem realizadas e acompanhar o percurso do motoboy até o destino.

O desenvolvimento do Mobile foi realizado apenas para os motoboys, contando com um sistema de recebimento de entregas há serem realizadas, um mapa que se atualiza conforme as entregas são aceitas e uma tela de perfil com as informações do motoboy.

Escopo do projeto

A Free Soul Food é um delivery de alimentação natural. Localizada na zona norte de São Paulo e atualmente atendem exclusivamente o público desta região.

A empresa possui constantes problemas com motoboys. Tentaram contratar serviços de empresas que prestam serviços de delivery, mas as empresas especializadas em entregas não atendem a região.

Junto com um grupo de empresários do segmento alimentício da zona norte, tiveram conhecimento que este é um problema comum a outros estabelecimentos.

Para solucionar este problema de forma coletiva, será desenvolvido uma plataforma que compartilhe entregas de motoboys. Esta plataforma permitirá que todos os restaurantes cadastrados compartilhem os seus motoboys, permitindo uma redução de custos e diminuição de tempo ocioso.

Funcionalidades

Localização

Localização
Raio do Restaurante: O sistema irá gerar um raio de distância para todos os restaurantes cadastrados.
Busca de motoboy: O sistema realiza uma busca de motoboys de acordo com o raio gerado do restaurante quregistrou a entrega.
Solicitação de motoboy: O sistema envia uma solicitação para o motoboy mais próximo para realizar a entrega do produto.
Localização do motoboy: O sistema disponibiliza um mapa para o restaurante poder acompanhar o motoboy no seu trajeto.
Finalização da entrega: O sistema envia uma notificação ao restaurante quando o motoboy chega ao destino final(endereço da entrega).

Mobile

Mobile
Login de motoboys: O sistema permite o login de motoboys já cadastrados.
Notificação: O sistema notifica para o motoboy mais próximo realizar a entrega do produto.
Rota: O sistema gera uma rota com base nas entregas que o motoboy aceitou.
Raio de solicitação: O sistema irá diminuir o raio de solicitações para recebimento de entregas do motoboy quando o mesmo estiver em movimento.

Administração

Administração
Cadastro de restaurantes: O sistema permite que o administrador registre os dados dos restaurantes.
Login ADM: O sistema terá um administrador padrão já cadastrado no banco de dados.
Cadastro de motoboys: O sistema permite que o administrador registre os dados dos motoboys.
Registro de entregas: O sistema permite que os restaurantes através dos funcionários registrem informações sobre a entrega (tamanho, fragilidade, endereço do cliente, preço).
